# Python Training Assignment

This repository contains my submissions for the Python Fundamentals. The codebase strictly follows Object-Oriented Principles (OOP) and Python best practices.

## Project Structure

* **`python_basic/`**: Contains core fundamental tracks including data structures, variables, operators, loops, modules, functions, oop and basic file handling.
* **`requirements.txt`**: Project dependencies and development tools.
* **`README.md`**: Description of the submission.

## Technical Standards Maintained

1. **PEP 8 Compliance**: Code is formatted strictly with 4 spaces for indentation.
2. **Modular Architecture**: Logic is separated into distinct, reusable functions and classes (Single Responsibility Principle).
3. **No Hardcoded Values**: Configuration values are defined as global constants (`UPPER_CASE_WITH_UNDERSCORES`) or handled via parameters.
4. **Documentation**: Clear docstrings (`"""`) and inline comments (`#`) explain the *why* behind complex logic.
5. **Type Hinting**: All functions utilize Python type hints for expected arguments and return types.
