# Python Advance 2

This folder contains the Python Advance 2 notebook assignments for the training course.

## Contents

- `assignment_1.ipynb` - NumPy basics and array operations
- `assignment_2.ipynb` - Pandas DataFrame creation and basic operations
- `assignment_3.ipynb` - Handling missing values with Pandas and NumPy
- `assignment_4.ipynb` - Grouping and aggregation with Pandas
- `assignment_5.ipynb` - Data visualization with Matplotlib
- `assignment_6.ipynb` - Data visualization with Seaborn and Matplotlib
- `assignment_7.ipynb` - Combined Pandas, Matplotlib, and Seaborn visualizations

## Requirements

The dependencies required to run these notebooks are listed in `requirements.txt`:

- `numpy>=1.26.0`
- `pandas>=2.1.0`
- `matplotlib>=3.8.0`
- `seaborn>=0.13.0`

## Setup Instructions

1. Create a virtual environment:

```bash
python -m venv venv
```

2. Activate the virtual environment:

- Windows PowerShell:

```powershell
venv\Scripts\Activate.ps1
```

- Windows Command Prompt:

```cmd
venv\Scripts\activate.bat
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Launch Jupyter Notebook or Jupyter Lab:

```bash
jupyter notebook
```

5. Open and run the notebook files from the browser interface.

## Notes

- `assignment_5.ipynb` and `assignment_7.ipynb` use `matplotlib` for charting.
- `assignment_6.ipynb` and `assignment_7.ipynb` use `seaborn` for advanced visualizations.
